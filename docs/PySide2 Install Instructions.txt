Instructions on how to install PySide2 5.9 for Python 3.6 (64-bit)

Get PySide 5.9 wheel 
PySide2-5.9-cp36-cp36m-win_amd64.whl
from
http://hansch.info/PySide2/

Install with

pip install PySide2-5.9-cp36-cp36m-win_amd64.whl

Make text file 'qt.conf' into the folder where the python.exe resides with the following contents:
[Paths]
Prefix = /Python36/Lib/site-packages/PySide2
Binaries = /Python36/Lib/site-packages/PySide2

If your site-packages folder is in another place, modify Prefix and Binaries lines in qt.conf to reflect that.

To make designer.exe work, copy 'qt.conf' file into the folder where qt.conf is

To make pyside2-uic work, clone pyside-setup project from https://code.qt.io/pyside/pyside-setup.git
Checkout branch 'dev'

Go to folder
\pyside-setup\sources\pyside2-tools\pyside2uic

Copy folders:
Compiler
port_v2
port_v3
widget-plugins

into folder
\Python36\Lib\site-packages\pyside2uic

At least \Compiler\ and \port_v3\ folders are required.
